Work done by data analyst

    Define the problems:

  • Determine the requirements whether they are dashboard, product analyst or reports
  • Create a plan
  • Communicate with the team

    Collect data:

  • Data is gathered from many sources. (Database backups, flat files, APIs)
  • Create an ETL procedure with the help of a coder. (Extract, Transform, Load)
  • Standardizing data and putting it all together.

    Clean data:

  • Since data is always messy, clean data makes data now easier to handle
  • Normalize and standardize data
  • Validation of data (check whether data looks like what they supposed to be)

    Set data up for reports/visualization:

  • Create views
  • Format data for its purpose
  • Filter data to narrow down your requirements.

    Create reports/visualization:

  • Automate reports in SQL
  • Connect data to data visualization tools (Or Python/R)
  • Make sure your reports/visualization meet the clientsâ€™ requirements.
